# GitHub Research: Open Source Bookkeeping Software

## Top Competitors Analyzed:

### 1. **Frappe/Books** (4,564 stars)
- Free accounting software
- Double-entry bookkeeping
- Invoicing, billing, payments
- Financial reports
- **What they lack:** AI automation, modern UX

### 2. **Beancount** (5,421 stars)
- Text-based accounting (Python)
- Double-entry from text files
- Command line focused
- **What they lack:** GUI, user-friendly, AI features

### 3. **GnuCash** (4,180 stars)
- Desktop app (C/C++)
- Double-entry accounting
- Personal & small business
- **What they lack:** Cloud, AI, modern web UX

### 4. **LedgerSMB** (530 stars)
- ERP system
- Double-entry accounting
- Web-based
- **What they lack:** AI, modern UX, mobile

### 5. **Dubbl** (9 stars)
- API-first alternative to Xero/QuickBooks
- Developer friendly
- Team-focused
- **What they lack:** Mature features, user base
