# BookKeepr AI

Premium autonomous bookkeeping automation for QuickBooks Online and Desktop.

## Pricing
- **Standard:** $199/mo (QBO only)
- **Silver:** $299/mo (QBO + QBD)
- **Gold:** $399/mo (Multi-company)

## Features (Roadmap)
- [ ] QBO Core Integration (Phase 1)
- [ ] AI Transaction Categorization (Phase 2)
- [ ] Dashboard & Analytics (Phase 3)
- [ ] Automated Reconciliation (Phase 4)
- [ ] QBD Support (Phase 5)
- [ ] Billing & Launch (Phase 6)

## Quick Start

```bash
# Setup
pip install -r requirements.txt
cp .env.example .env

# Configure your Intuit credentials in .env

# Run
flask run
```
