## 📋 HOW TO IMPORT REAL DATA

### Option 1: QuickBooks Connect
1. Go to Settings → Integrations
2. Click "Connect QuickBooks"
3. Authorize BookKeepr
4. Select company
5. Sync transactions

### Option 2: CSV Upload
1. Go to Transactions → Import
2. Download template (optional)
3. Upload CSV file
4. Map columns
5. Review and confirm

### Option 3: Manual Entry
1. Go to Transactions → Add New
2. Fill in details
3. Categorize
4. Save
